# Tier 1: Trailwise Toolkit

Production-ready finance ops tools. These are the skills you run when you need a working solution today.

## Skills

| Skill | Problem it solves |
|-------|-------------------|
| `invoice-reconciliation` | Match invoices to POs, flag discrepancies |
| `budget-variance-tracker` | Real-time budget vs actual with alerts |
| `cash-flow-forecaster` | Project cash flow, predict shortfalls |
| `ar-aging-report` | Accounts receivable aging with auto-categorization |
| `ap-aging-report` | Accounts payable aging with payment scheduling |
| `payment-app-generator` | Generate AIA-style payment applications |
| `retention-tracker` | Track retainage/retention across projects |
| `change-order-cost-impact` | Calculate cost impact of change orders |
| `expense-categorizer` | Auto-categorize expenses by cost code |
| `vendor-payment-tracker` | Track vendor payments, terms, discounts |
| `payroll-calculator` | Calculate payroll with burden and labor cost |
| `unit-cost-database` | Manage unit cost database with regional factors |
| `estimate-builder` | Build estimates from templates and historical data |
| `revenue-recognition` | % complete revenue recognition calculations |
| `financial-kpi-dashboard` | Interactive KPI dashboard from live data |
