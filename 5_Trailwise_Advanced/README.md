# Tier 5: Trailwise Advanced

Predictive analytics and ML skills for finance operations. These use historical data to forecast outcomes, detect anomalies, and score risk.

## Skills

| Skill | What it predicts |
|-------|-----------------|
| `cost-overrun-prediction` | Which projects will exceed budget (ML classification) |
| `invoice-anomaly-detector` | Unusual invoices that may be fraud or errors |
| `cash-flow-forecast-ml` | ML-based cash flow projection (better than linear) |
| `vendor-risk-scorer` | Vendor risk score based on payment history, volume, terms |
| `budget-forecast-ml` | Predict final project cost from early-stage data |
