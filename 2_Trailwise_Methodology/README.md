# Tier 2: Trailwise Methodology

Skills mapped to finance operations principles. These teach the *why* behind the automation — understanding methodology before deploying tools.

## Skills

| Skill | What it teaches |
|-------|----------------|
| `data-source-audit` | Inventory all financial data sources, map flows, find silos |
| `data-silo-detection` | Identify disconnected data that blocks automation |
| `data-quality-check` | Validate financial data quality (completeness, accuracy, consistency) |
| `accrual-vs-cash` | Analyze when to use accrual vs cash basis for project accounting |
| `percentage-complete` | % complete revenue recognition methodology and calculations |
| `workflow-mapping` | Map financial workflows end-to-end, identify bottlenecks |
| `compliance-checklist` | Financial compliance requirements by industry and jurisdiction |
