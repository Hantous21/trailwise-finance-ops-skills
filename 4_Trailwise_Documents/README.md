# Tier 4: Trailwise Documents

Document generation skills — formatted output (payment applications, AR aging workbooks, financial reports) built from your structured data. Output is print-ready (PDF) or directly shareable with the bank, owner, or CFO.

## Shipped

| Skill | Description |
|---|---|
| `payment-app-html` | Render a structured G702/G703 pay app as a self-contained, print-ready HTML document. |
| `ar-aging-excel` | Build an AR aging Excel workbook (summary + chart + invoice detail) from an invoice export. |

## Planned

| Skill | Description |
|---|---|
| `lien-waiver-pdf` | Generate conditional / unconditional lien waiver PDFs from a sub list and draw record. |
| `financial-report-pdf` | Render the monthly financial summary (cash, AR, AP, job cost) as a PDF. |
| `wip-pdf` | Render the WIP schedule with over/under billings as a print-ready PDF. |

Until additional skills ship here, the closest `1_Trailwise_Toolkit/` skills that produce the underlying data are `payment-app-generator` and `ar-collections-automation`.
