# Tier 4: Trailwise Documents

Document generation skills. These produce formatted output — payment applications, lien waivers, financial reports, KPI dashboards — from your structured data.

## Skills

| Skill | What it generates |
|-------|------------------|
| `payment-app-pdf` | AIA-style payment application (PDF) |
| `lien-waiver-pdf` | Conditional/unconditional lien waivers (PDF) |
| `financial-report-pptx` | Stakeholder financial presentations (PowerPoint) |
| `estimate-excel` | Formatted estimate spreadsheets with formulas |
| `ar-aging-excel` | AR aging report with charts |
| `cash-flow-statement` | Cash flow statement (Excel/PDF) |
| `kpi-dashboard-html` | Interactive KPI dashboard (HTML/JS) |
| `budget-report-excel` | Budget vs actual report with variance charts |
