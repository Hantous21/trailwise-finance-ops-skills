# Tier 6: Trailwise Contractor Operations

Field and project controls skills. These are the workflows that protect the
job before, during, and after the work — submittals, RFIs, daily logs,
schedule, and the paperwork that closes out a draw.

## Skills

| Skill | What it does |
|-------|--------------|
| `daily-field-report` | Compile daily field reports with labor hours, delays, safety flags |
| `rfi-management` | Track RFIs, triage by priority, flag overdue items |
| `schedule-delay-analyzer` | CPM critical path analysis, delay simulation, float calculation |
| `submittal-tracker` | Track submittals, calculate required dates, flag late items |
| `lien-waiver-tracker` | Track lien waiver collection, exposure, and draw blockers |
| `certified-payroll-report` | WH-347-style weekly certified payroll with Davis-Bacon check |
