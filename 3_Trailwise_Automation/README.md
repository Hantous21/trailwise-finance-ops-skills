# Tier 3: Trailwise Automation

Workflow automation skills using n8n, Zapier, and custom scripts. These connect your tools and automate the manual data movement between them.

## Skills

| Skill | What it automates |
|-------|------------------|
| `n8n-invoice-approval` | Auto-route invoices for approval via n8n |
| `n8n-payment-reminders` | Send payment reminders to clients and vendors |
| `n8n-budget-alerts` | Budget threshold alerts via Slack/Teams/email |
| `n8n-recurring-reports` | Auto-generate and distribute recurring financial reports |
| `n8n-expense-import` | Import expenses from bank API into accounting system |
| `telegram-finance-bot` | Telegram bot for quick finance queries and approvals |
